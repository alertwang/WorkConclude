让svn在使用checkout命令时不含根目录

让svn在checkout时不含根目录,使用图形模式时很方便,使用命令行模式时需要注意：

不含根目录的意思：比如我要checkout  http://192.168.9.12:8443/svn/保卫萝卜/ 下面的所有文件，但是不包括保卫萝卜文件夹,
我们可以在路径文件夹后面打个空格，再加个“.”就O了
示例: 
set svn_url=http://192.168.9.12:8443/svn/保卫萝卜/ .
set svn_user=mojitu
set svn_ps=123456
svn checkout %svn_url% --username %svn_user% --password %svn_ps%
